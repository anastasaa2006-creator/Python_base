Приложение для работы с данными фермерских рынков
ЛР7
Как запустить:
1. Файл `Export.csv` должен находиться в той же папке, что и программа `ЛР7.py`
2. Запуск через консоль: python ЛР7.py

Команды
После запуска программа предложит ввести команду, доступные представлены в скобках: Command ('loc', 'zip', 'dist', 'end') =>;
- `list` — список рынков (по 5 на страницу)
- `view` — подробная информация о рынке
- `search` — поиск по городу и штату
- `zip` — поиск по почтовому индексу
- `dist` — поиск в радиусе (мили)
- `add` — добавить отзыв
- `show` — показать отзывы
- `sort` — сортировка
- `del` — удалить отзыв
- `end` — выход

Файлы
- `Export.csv` — данные о рынках
- `reviews.json` — отзывы (создаётся автоматически)
