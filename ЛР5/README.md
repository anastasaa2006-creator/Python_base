Игра "Жизнь" (Game of Life)

Программа:
- Читает начальную конфигурацию из файла `input_life.csv`
- Выполняет 10 поколений моделирования
- Сохраняет каждое поколение в текстовый файл вида `output_000.txt`
- Сохраняет каждое поколение в PNG-изображение вида `output_000.png`

Формат входного файла:

**Первая строка** — размеры поля:  `rows cols` (количество строк и столбцов)

**Следующие строки** — матрица из `0` и `1`:
- `0` — мертвая клетка
- `1` — живая клетка
