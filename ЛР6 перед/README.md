ЛР6
Как запустить:
1. Файл `zip_codes_states.csv` должен находиться в той же папке, что и программа `ЛР6.py`
2. Запуск через консоль: python ЛР6.py

О программе:
После запуска программа предложит ввести команду, доступные представлены в скобках: Command ('loc', 'zip', 'dist', 'end') =>
loc - поиск информации по ZIP-коду
zip - поиск ZIP-кодов по городу и штату
dist - расчет расстояния между двумя ZIP-кодами
end - выход из программы (Done)


- ЛР6.py - основной код 
- zip_codes_states.csv - база данных (обязательно)
